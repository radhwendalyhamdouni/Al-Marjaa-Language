## الملخص | Abstract

تقدم هذه الورقة **لغة المرجع**، أول لغة برمجة عربية متكاملة مع الذكاء الاصطناعي. تم تصميم اللغة لتعزيز الإنتاجية للمتحدثين باللغة العربية من خلال توفير بنية لغوية طبيعية تدعم الكتابة من اليمين إلى اليسار والأرقام العربية والكلمات المفتاحية العربية. تشمل المساهمات الرئيسية: (1) محلل معجمي ونحوي كامل يدعم العربية، (2) مفسر شجري مع مكتبة دوال أصلية واسعة، (3) آلة افتراضية bytecode مع 5 طبقات تحسين JIT، (4) ميزة **Vibe Coding** الثورية للبرمجة باللغة الطبيعية، (5) تكامل كامل مع عمليات الذكاء الاصطناعي. أظهرت الاختبارات نسبة نجاح 100% في 343 اختبار، مع أداء مقارن للغات المفسرة المماثلة.

This paper presents **Al-Marjaa**, the first AI-native Arabic programming language. The language is designed to enhance productivity for Arabic speakers by providing a natural linguistic structure supporting right-to-left writing, Arabic numerals, and Arabic keywords. Key contributions include: (1) a complete lexer and parser supporting Arabic, (2) a tree-walking interpreter with extensive native function library, (3) a bytecode VM with 5-tier JIT optimization, (4) the revolutionary **Vibe Coding** feature for natural language programming, (5) full integration with AI operations. Tests showed a 100% pass rate across 343 tests, with performance comparable to similar interpreted languages.

## 1. المقدمة | Introduction

### 1.1 الخلفية والأهمية

تمثل البرمجة باللغة الإنجليزية عائقاً رئيسياً للمتحدثين باللغة العربية، الذين يزيد عددهم عن 400 مليون نسمة. تظهر الدراسات أن المبرمجين العرب يواجهون تحديات إضافية تشمل:

- **الحمل المعرفي المزدوج**: ترجمة المفاهيم بين اللغتين
- **أخطاء الكتابة**: الخلط بين أحرف RTL و LTR
- **صعوبة التصحيح**: رسائل خطأ غير مألوفة
- **منحنى تعلم حاد**: تأخر الإنتاجية

### 1.2 الأهداف

1. تصميم لغة برمجة مع بنية نحوية عربية طبيعية
2. تكامل الذكاء الاصطناعي لتقليل الفجوة بين الفكرة والتنفيذ
3. توفير بيئة تطوير متكاملة مع دعم IDE
4. ضمان أداء مقبول مع قابلية التوسع

### 1.3 المساهمات

| المساهمة | الوصف |
|----------|-------|
| المحلل المعجمي العربي | دعم كامل للكلمات المفتاحية والأرقام العربية |
| المحلل النحوي | Recursive Descent مع استرداد الأخطاء |
| المفسر | Tree-walking مع مكتبة دوال شاملة |
| آلة Bytecode | Stack-based VM مع JIT |
| Vibe Coding | تحويل النص العربي إلى كود |
| تكامل AI | Tensor, NeuralNetwork, Autograd |

fn convert_arabic_digits(s: &str) -> String {
    s.chars().map(|c| match c {
        '٠' => '0', '١' => '1', '٢' => '2',
        '٣' => '3', '٤' => '4', '٥' => '5',
        '٦' => '6', '٧' => '7', '٨' => '8',
        '٩' => '9', _ => c,
    }).collect()
}

### 3.3 المحلل النحوي

#### قواعد النحو

```
program      → statement* EOF
statement    → varDecl | constDecl | funcDecl | classDecl
             | ifStmt | whileStmt | forStmt | returnStmt
             | exprStmt | block

varDecl      → "متغير" IDENTIFIER ("=" expression)? "؛"
funcDecl     → "دالة" IDENTIFIER "(" parameters? ")" block
ifStmt       → "إذا" expression block ("وإلا" block)?
whileStmt    → "طالما" expression block
forStmt      → "لكل" IDENTIFIER "في" expression block
```

### 3.4 آلة Bytecode

#### التعليمات

| التعليمة | الرمز | الوصف |
|----------|-------|-------|
| OP_CONST | 0x01 | تحميل ثابت |
| OP_ADD | 0x02 | جمع |
| OP_SUB | 0x03 | طرح |
| OP_MUL | 0x04 | ضرب |
| OP_DIV | 0x05 | قسمة |
| OP_CALL | 0x10 | استدعاء دالة |
| OP_RETURN | 0x11 | إرجاع |
