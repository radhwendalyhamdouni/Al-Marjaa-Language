## المقدمة

لغة المرجع هي أول لغة برمجة عربية متكاملة مع الذكاء الاصطناعي. تتميز بـ:

- **دعم عربي كامل**: كلمات مفتاحية عربية، أرقام عربية، كتابة من اليمين لليسار
- **تكامل AI**: ميزة Vibe Coding للبرمجة باللغة الطبيعية
- **أنواع متقدمة**: Tensor, NeuralNetwork, GPU operations
- **أدوات متكاملة**: REPL، LSP، مدير حزم

## التثبيت

### المتطلبات

- Rust 1.70 أو أحدث
- نظام تشغيل: Linux, macOS, Windows

```bash

cd Al-Marjaa-Language

cargo build --release
```

## العمليات الحسابية

### العمليات الأساسية

```almarjaa
متغير أ = 10؛
متغير ب = 3؛

اطبع(أ + ب)؛    // 13 (جمع)
اطبع(أ - ب)؛    // 7 (طرح)
اطبع(أ * ب)؛    // 30 (ضرب)
اطبع(أ / ب)؛    // 3.33 (قسمة)
اطبع(أ % ب)؛    // 1 (باقي)
اطبع(أ ^ ب)؛    // 1000 (أس)
```

### عمليات المقارنة

```almarjaa
اطبع(5 == 5)؛    // صحيح
اطبع(5 != 3)؛    // صحيح
اطبع(5 > 3)؛     // صحيح
اطبع(5 < 3)؛     // خطأ
اطبع(5 >= 5)؛    // صحيح
اطبع(5 <= 3)؛    // خطأ
```

### العمليات المنطقية

```almarjaa
اطبع(صحيح و خطأ)؛    // خطأ
اطبع(صحيح أو خطأ)؛   // صحيح
اطبع(ليس صحيح)؛      // خطأ
```

## الجمل الشرطية

### إذا البسيطة

```almarjaa
متغير عمر = 20؛

إذا عمر >= 18 {
    اطبع("بالغ")؛
```

### إذا-وإلا

```almarjaa
متغير درجة = 75؛

إذا درجة >= 90 {
    اطبع("ممتاز")؛
} وإلا إذا درجة >= 70 {
    اطبع("جيد")؛
```

### المعامل الثلاثي

```almarjaa
متغير عدد = 10؛
متغير النتيجة = عدد > 5 ? "كبير" : "صغير"؛

```

## الحلقات

### حلقة طالما (While)

```almarjaa
متغير عداد = 0؛

طالما عداد < 5 {
    اطبع(عداد)؛
    عداد += 1؛
}
```

### حلقة لكل (For)

```almarjaa
// على نطاق
لكل رقم في مدى(1، 6) {
    اطبع(رقم)؛    // 1, 2, 3, 4, 5
}

// على قائمة
متغير أسماء = ["أحمد"، "محمد"، "علي"]؛
لكل اسم في أسماء {
    اطبع("مرحباً " + اسم)؛
}
```

### حلقة كرر (Repeat)

```almarjaa
كرر 5 مرات {
    اطبع("مرحباً")؛
}
```

### التحكم في الحلقة

```almarjaa
// break - الخروج من الحلقة
طالما صحيح {
    متغير إدخال = اقرأ()؛
    إذا إدخال == "خروج" {
        أوقف؛    // break
    }
}

// continue - تخطي التكرار الحالي
لكل رقم في مدى(1، 11) {
    إذا رقم % 2 == 0 {
        تابع؛    // continue
    }
    اطبع(رقم)؛    // 1, 3, 5, 7, 9
}
```

## الدوال

### تعريف الدوال

```almarjaa
// دالة بسيطة
دالة ترحيب() {
    اطبع("مرحباً!")؛
}

// دالة مع معاملات
دالة جمع(أ، ب) {
    أرجع أ + ب؛
}

// دالة مع قيم افتراضية
دالة تحية(الاسم = "زائر") {
    أرجع "مرحباً " + الاسم؛
}
```

### استدعاء الدوال

```almarjaa
ترحيب()؛                    // مرحباً!
اطبع(جمع(5، 3))؛            // 8
اطبع(تحية())؛               // مرحباً زائر
اطبع(تحية("أحمد"))؛         // مرحباً أحمد
```

### الدوال السهمية (Lambda)

```almarjaa
// تعبير lambda
متغير مضاعف = (س) => س * 2؛
اطبع(مضاعف(5))؛             // 10

// كدالة معامل
متغير أرقام = [1، 2، 3، 4، 5]؛
مت_var مزدوج = خريطة(أرقام، (س) => س * 2)؛
اطبع(مزدوج)؛                // [2, 4, 6, 8, 10]
```

### الإغلاق (Closures)

```almarjaa
دالة صنع_عداد() {
    متغير عدد = 0؛
    أرجع () => {
        عدد += 1؛
        أرجع عدد؛
    }؛
}

متغير العداد = صنع_عداد()؛
اطبع(العداد())؛    // 1
اطبع(العداد())؛    // 2
اطبع(العداد())؛    // 3
```

## القوائم والقواميس

### عمليات القوائم

```almarjaa
متغير قائمة = [1، 2، 3، 4، 5]؛

// الإضافة
قائمة.أضف(6)؛              // [1, 2, 3, 4, 5, 6]
قائمة.أدخل(0، 100)؛       // [100, 1, 2, 3, 4, 5, 6]

// الحذف
قائمة.احذف(0)؛            // يحذف العنصر الأول
قائمة.انزع()؛             // يحذف الأخير

// البحث
اطبع(قائمة.بحث(3))؛       // الفهرس
اطبع(قائمة.يحتوي(5))؛     // صحيح/خطأ

// الترتيب
قائمة.رتب()؛              // تصاعدي
قائمة.اعكس()؛             // عكس الترتيب

// الطول
اطبع(طول(قائمة))؛
```

### القوائم الاستيعابية

```almarjaa
// مربعات الأرقام من 1 إلى 10
متغير مربعات = [س * س لكل س في مدى(1، 11)]؛

// الأرقام الزوجية فقط
متغير زوجية = [س لكل س في مدى(1، 11) إذا س % 2 == 0]؛

// تحويل نصوص
متغير أسماء = ["أحمد"، "محمد"، "علي"]؛
متغير تحيات = ["مرحباً " + اسم لكل اسم في أسماء]؛
```

## البرمجة الكائنية

### تعريف الصنف

```almarjaa
صنف حيوان {
    متغير الاسم؛
    متغير العمر؛
    
    // البناء
    دالة حيوان(الاسم، العمر) {
        هذا.الاسم = الاسم؛
        هذا.العمر = العمر؛
    }
    
    // دالة
    دالة صوت() {
        أرجع "صوت عام"؛
    }
    
    دالة معلومات() {
        أرجع هذا.الاسم + " - " + هذا.العمر + " سنة"؛
    }
}
```

### الوراثة

```almarjaa
صنف كلب: حيوان {
    دالة صوت() {
        أرجع "نباح!"؛
    }
    
    دالة حراسة() {
        اطبع(هذا.الاسم + " يراقب المنزل")؛
    }
}

// استخدام
متغير كلبي = جديد كلب("بوبي"، 3)؛
اطبع(كلبي.صوت())؛        // نباح!
اطبع(كلبي.معلومات())؛    // بوبي - 3 سنة
كلبي.حراسة()؛            // بوبي يراقب المنزل
```

## Vibe Coding

### ما هو Vibe Coding؟

Vibe Coding هي ميزة تتيح لك كتابة البرامج باستخدام اللغة العربية الطبيعية. تقوم الذكاء الاصطناعي بتحويل أوامرك النصية إلى كود قابل للتنفيذ.

### تفعيل Vibe Coding

```almarjaa
// في REPL، اكتب:
ذكاء

// أو من سطر الأوامر:
almarjaa --vibe
```

### أمثلة على Vibe Coding

#### المتغيرات

```
الإدخال: أنشئ متغير اسم يساوي أحمد
المُنتج: متغير اسم = "أحمد"؛
```

#### الشروط

```
الإدخال: إذا كان العمر أكبر من 18 اطبع بالغ
المُنتج: إذا العمر > 18 { اطبع("بالغ")؛ }
```

#### الدوال

```
الإدخال: اكتب دالة تجمع رقمين
المُنتج: دالة اجمع(أ، ب) { أرجع أ + ب؛ }
```

#### الحلقات

```
الإدخال: كرر طباعة مرحبا 5 مرات
المُنتج: لكل _ في مدى(5) { اطبع("مرحبا")؛ }
```

### نصائح لاستخدام Vibe Coding

1. **كن واضحاً**: استخدم عبارات دقيقة وواضحة
2. **حدد الأسماء**: "أنشئ متغير اسم" أفضل من "أنشئ متغير"
3. **اذكر القيم**: "يساوي 10" أفضل من "بقيمة عشرة"
4. **استخدم المصطلحات**: "دالة"، "حلقة"، "شرط"

## نظام واجهات المستخدم

### ما هو نظام واجهات المستخدم؟

نظام واجهات المستخدم في لغة المرجع هو إطار عمل متكامل لبناء تطبيقات واجهات المستخدم باستخدام كلمات مفتاحية عربية بسيطة. يتضمن:

- **نظام التخطيط التلقائي**: صف، عمود، شبكة، مرن
- **ربط البيانات التلقائي**: تحديث الواجهة تلقائياً عند تغيير البيانات
- **المكونات القابلة لإعادة الاستخدام**: بناء مكونات مخصصة
- **التصميم المتجاوب**: تكييف الواجهة حسب حجم الشاشة
- **نظام الثيمات**: تخصيص الألوان والخطوط
- **الرسوم البيانية**: إنشاء رسوم بيانية تفاعلية
- **النوافذ المنبثقة**: حوارات وتنبيهات

### نظام التخطيط

#### الصف (Row)

```almarjaa
// تخطيط أفقي
صف {
    فجوة: 10،
    محاذاة: "وسط"،
    
    زر("السابق")،
    نص("الصفحة 1")،
    زر("التالي")،
}
```

#### العمود (Column)

```almarjaa
// تخطيط عمودي
عمود {
    فجوة: 15،
    
    نص("العنوان") { حجم: 24، وزن_الخط: "غامق" }،
    نص("الوصف") { لون: "#666666" }،
    زر("ابدأ الآن")،
}
```

#### الشبكة (Grid)

```almarjaa
// تخطيط شبكي
شبكة {
    أعمدة: 3،
    صفوف: 2،
    فجوة: 20،
    
    بطاقة("منتج 1")،
    بطاقة("منتج 2")،
    بطاقة("منتج 3")،
    بطاقة("منتج 4")،
    بطاقة("منتج 5")،
    بطاقة("منتج 6")،
}
```

### ربط البيانات

#### القيم القابلة للملاحظة

```almarjaa
// إنشاء قيمة قابلة للملاحظة
متغير العدد = قابل_للملاحظة(0)؛

// عند تغيير القيمة، تتحدث جميع المكونات المرتبطة
العدد.قيمة = 10؛
```

#### الربط الأحادي

```almarjaa
// الربط من المصدر إلى الواجهة فقط
نص(العدد.قيمة) {
    // النص يتحدث تلقائياً
}
```

#### الربط الثنائي

```almarjaa
// الربط في الاتجاهين
إدخال {
    ربط: العدد.قيمة،
    // التغيير في الإدخال يحدث القيمة
    // وتغيير القيمة يحدث الإدخال
}
```

#### القيم المحسوبة

```almarjaa
// قيمة تُحسب تلقائياً من قيم أخرى
متغير المجموع = محسوب(() => {
    أرجع العدد.قيمة * 2؛
})؛

نص(المجموع.قيمة)؛  // يتحدث تلقائياً عند تغيير العدد
```

### المكونات

#### المكونات الأساسية

```almarjaa
// زر
زر("اضغط هنا") {
    لون_الخلفية: "#3498db"،
    لون_النص: "#ffffff"،
    نقر: () => اطبع("تم الضغط!")،
}

// نص
نص("مرحباً بالعالم") {
    حجم: 18،
    لون: "#2c3e50"،
    وزن_الخط: "غامق"،
}

// إدخال
إدخال {
    placeholder: "أدخل اسمك"،
    ربط: الاسم.قيمة،
    تغيير: (القيمة) => اطبع(القيمة)،
}

// بطاقة
بطاقة {
    ظل: "متوسط"،
    حشو: 20،
    
    عمود {
        نص("عنوان البطاقة")،
        نص("وصف البطاقة")،
    }،
}
```

#### إنشاء مكون مخصص

```almarjaa
// تعريف مكون
مكون بطاقة_منتج {
    خصائص: [الاسم، السعر، الصورة]،
    
    بطاقة {
        صف {
            صورة(الصورة) { عرض: 80، ارتفاع: 80 }،
            عمود {
                نص(الاسم) { وزن_الخط: "غامق" }،
                نص(السعر + " ر.س") { لون: "#27ae60" }،
            }،
        }،
    }،
```
