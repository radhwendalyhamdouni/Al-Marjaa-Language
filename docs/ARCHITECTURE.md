## 📋 نظرة عامة

لغة المرجع هي لغة برمجة عربية متكاملة مع ذكاء اصطناعي، صُممت لتكون:
- **عربية بامتياز**: بناء لغوي ونحوي عربي أصيل
- **سهلة التعلم**: قواعد بسيطة وواضحة
- **متكاملة مع AI**: دعم Vibe Coding والتحويل من النص الطبيعي
- **عالية الأداء**: JIT Compiler و Parallel GC
