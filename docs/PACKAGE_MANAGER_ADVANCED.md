# نظام الحزم المتقدم - لغة المرجع

## نظرة عامة

نظام حزم عربي متكامل وفريد من نوعه يوفر تجربة سلسة للمطورين مع ميزات متقدمة لا توجد في أنظمة الحزم الأخرى.

---

## المكونات الأساسية

### 1. ملف التعريب العربي (`مشروع.toml`)

```toml
# ═══════════════════════════════════════════════════════════════════
# ملف تعريف المشروع - لغة المرجع
# ═══════════════════════════════════════════════════════════════════

اسم = "مشروعي"
إصدار = "1.0.0"
وصف = "مشروع جديد بلغة المرجع"
مؤلف = "المطور"
رخصة = "MIT"

[تبعيات]
json = "1.0.0"
http = "^2.1.0"

[تبعيات_تطوير]
اختبارات = "1.0.0"
```

### 2. الأوامر العربية

| الأمر | الوصف |
|-------|-------|
| `almarjaa أنشئ <اسم>` | إنشاء مشروع جديد |
| `almarjaa ثبّت` | تثبيت جميع التبعيات |
| `almarjaa أضف <حزمة>` | إضافة حزمة جديدة |
| `almarjaa أزل <حزمة>` | إزالة حزمة |
| `almarjaa حدّث` | تحديث الحزم |
| `almarjaa ابحث <استعلام>` | البحث عن حزم |
| `almarjaa انشر` | نشر حزمة |
| `almarjaa قائمة` | عرض الحزم المثبتة |
| `almarjaa أمان` | فحص أمني |

## الميزات المتقدمة

### 🏅 نظام الشارات والتقييم

كل حزمة تحصل على تقييم شامل بناءً على معايير متعددة:

#### معايير التقييم:
- **الأمان** (25%): خلو من الثغرات، التحقق من التبعيات
- **الجودة** (25%): الاختبارات، التغطية، CI/CD
- **الشعبية** (20%): التنزيلات، النجوم، الفروع
- **الصيانة** (15%): التحديثات، المساهمون
- **التوثيق** (15%): README، CHANGELOG، التعليقات

#### الشارات المتاحة:

| الشارة | الأيقونة | الوصف |
|--------|----------|-------|
| آمن | 🔒 | خالية من الثغرات الأمنية |
| موثق | ✅ | مؤلف موثوق ومتحقق |
| مختبر | 🧪 | اختبارات شاملة |
| موثق | 📚 | توثيق شامل |
| بسيط | 📦 | تبعيات قليلة |
| شائع | ⭐ | محبوب من المطورين |
| مُصان | 🔧 | محدث بانتظام |
| عربي | 🌙 | مكتوب بالكامل بالعربية |

#### مستويات الحزم:
- 💎 **بلاتيني**: 90-100 نقطة
- 🥇 **ذهبي**: 75-89 نقطة
- 🥈 **فضي**: 60-74 نقطة
- 🥉 **برونزي**: 40-59 نقطة
- 📦 **أساسي**: أقل من 40 نقطة

### 📦 المستودع اللامركزي

نظام مستودع موزع يعتمد على GitHub:

- **الفهرس المركزي**: `github.com/almarjaa-lang/registry`
- **CDN عالمي**: jsDelivr، GitHub CDN
- **التوقيع الرقمي**: التحقق من سلامة الحزم
- **التخزين المؤقت الذكي**: كاش محلي مع TTL

### 📚 التوثيق التلقائي

توليد توثيق Markdown تلقائياً من الكود المصدري:

```mrj
/// دالة لتحية المستخدم
/// تأخذ اسم المستخدم وتعيد رسالة ترحيب
دالة حيّ(الاسم: نص) -> نص {
    أرجع "مرحبا " + الاسم
}
```

يُنتج:

```markdown
### `حيّ`

تحية المستخدم

**المعاملات:**
- `الاسم`: `نص`

**القيمة المرجعة:** `نص`
```

### 👥 نظام السمعة والمؤلفين

#### مستويات المؤلفين:

| المستوى | الأيقونة | النقاط المطلوبة |
|---------|----------|-----------------|
| مبتدئ | 🌱 | 0 |
| مساهم | 🤝 | 100 |
| مطور | 💻 | 500 |
| خبير | ⭐ | 2,000 |
| محترف | 🏆 | 10,000 |
| أسطورة | 👑 | 50,000 |

#### شارات المؤلفين:
- 🎁 **البداية**: نشر أول حزمة
- 📦 **مُنتِج**: نشر 10 حزم
- 🌟 **مشهور**: 1000 متابع
- 🤝 **مُفيد**: 50 مساهمة مجتمع
- 🐛 **صياد الأخطاء**: 20 تقرير خطأ
- ✅ **موثق**: تم التحقق من الحساب
- ⭐ **صانع النجوم**: 1000 نجمة على الحزم
- 👑 **ملك التنزيلات**: 10000 تنزيل

### 🔄 الإصدارات الدلالية والـ CI/CD

#### الإصدارات الدلالية:

```rust
// تحليل رسائل commit تلقائياً
"feat: ميزة جديدة"      // -> minor bump (1.0.0 -> 1.1.0)
"fix: إصلاح خطأ"        // -> patch bump (1.0.0 -> 1.0.1)
"breaking!: تغيير جوهري" // -> major bump (1.0.0 -> 2.0.0)
```

#### التكامل المستمر:

```yaml
# .github/workflows/ci.yml
name: CI

on:
  push:
    branches: [ main, develop ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
    - name: فحص التنسيق
    - name: فحص Lint
    - name: فحص الأمان
    - name: تشغيل الاختبارات
    - name: البناء
```

## أمثلة استخدام

### إنشاء مشروع جديد

```bash
$ almarjaa أنشئ تطبيق_الويب

🚀 تهيئة مشروع جديد: تطبيق_الويب
   ✅ تم إنشاء المشروع بنجاح
   📁 تطبيق_الويب/
   ├── مشروع.toml
   ├── مصدر/
   │   └── رئيسي.mrj
   └── حزم/
```

### إضافة حزمة

```bash
$ almarjaa أضف json

📦 إضافة حزمة 'json'...
   📌 الإصدار: 1.0.0
   🔒 فحص أمني... ✅ آمن
   ✅ تمت الإضافة بنجاح
```

```bash
$ almarjaa ابحث http

🔍 البحث عن 'http'...

   📦 نتائج البحث (3):
   
   ┌─ http v2.1.0
   │  مكتبة طلبات HTTP الشبكية
   │  ⬇️ 15000 تحميل | ⭐ 200 نجمة

   ┌─ https v1.0.0
   │  امتداد آمن لـ HTTP
```
