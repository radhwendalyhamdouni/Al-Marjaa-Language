# دعم ONNX في لغة المرجع

## نظرة عامة

يوفر دعم ONNX (Open Neural Network Exchange) في لغة المرجع إمكانية:

- تحميل نماذج الذكاء الاصطناعي من أي إطار عمل (PyTorch, TensorFlow, scikit-learn)
- تشغيل الاستدلال بكفاءة عالية
- تصدير النماذج المدربة بصيغة ONNX
- التكامل السلس مع الأنواع الأصلية للغة

## التثبيت

لتفعيل دعم ONNX، قم بتجميع المشروع مع ميزة `onnx`:

```bash
cargo build --features onnx
```

أو لتفعيل جميع الميزات:

```bash
cargo build --features full
```

## الاستخدام الأساسي

### 1. تحميل نموذج ONNX

```مرجع
# تحميل نموذج من ملف
نموذج = أونكس.حمّل("model.onnx")

# عرض معلومات النموذج
اطبع(نموذج.معلومات)
```

### 2. إنشاء الموترات

```مرجع
# موتر من قيم
موتر أ = موتر([1, 2, 3, 4], شكل=[2, 2])

# موتر صفري
موتر ب = موتر.أصفار([3, 3])

# موتر من واحد
موتر ج = موتر.واحدات([2, 4])

# موتر عشوائي
موتر د = موتر.عشوائي([1, 3, 224, 224])
```

### 3. تشغيل الاستدلال

```مرجع
# تحضير المدخلات
موتر مدخل = موتر.عشوائي([1, 3, 224, 224])

# تشغيل الاستدلال
نتيجة = نموذج.استدل({"input": مدخل})

# الحصول على المخرجات
مخرج = نتيجة["output"]
```

### 4. تصدير نموذج جديد

```مرجع
# إنشاء شبكة عصبية
شبكة = أونكس.شبكة_جديدة("مصنف")

# إضافة طبقات
شبكة.أضف_طبقة({
    "نوع": "كثيف",
    "مدخل": 784,
    "مخرج": 128,
    "تنشيط": "ريلو"
})

شبكة.أضف_طبقة({
    "نوع": "كثيف",
    "مدخل": 128,
    "مخرج": 10,
    "تنشيط": "سوفت_ماكس"
})

# تصدير
شبكة.صدّر("classifier.onnx")
```

## الكلمات المفتاحية

### كلمات ONNX الأساسية

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| أونكس | onnx | الوصول لوحدة ONNX |
| نموذج | model | تعريف نموذج |
| حمّل | load | تحميل نموذج |
| احفظ | save | حفظ نموذج |
| استدل | infer | تشغيل الاستدلال |
| موتر | tensor | تعريف موتر |
| شكل | shape | شكل الموتر |
| طبقة | layer | طبقة عصبية |

### طبقات الشبكة

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| كثيف | dense | طبقة كثيفة/خطية |
| التفاف | conv | طبقة التفاف |
| تجميع | pool | طبقة تجميع |
| طبع | normalize | طبقة تسوية |
| إسقاط | dropout | طبقة إسقاط |
| تسوية | flatten | طبقة تسوية الأبعاد |
| إعادة_تشكيل | reshape | إعادة تشكيل الموتر |

### دوال التنشيط

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| تنشيط | activation | تفعيل دالة التنشيط |
| سوفت_ماكس | softmax | دالة Softmax |
| ريلو | relu | دالة ReLU |
| سيجمويد | sigmoid | دالة Sigmoid |

### التدريب

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| دفعة | batch | حجم الدفعة |
| درّب | train | تدريب النموذج |
| توقع | predict | التنبؤ |
| محسّن | optimizer | محسّن التدريب |
| خسارة | loss | دالة الخسارة |

### دوال تنشيط

- Elu, Selu, Softplus, Softsign

### عمليات المصفوفات
- Reshape, Transpose, Flatten, Squeeze, Unsqueeze
- Concat, Split, Slice, Gather, Scatter

### الالتفاف
- Conv, ConvTranspose, MaxPool, AveragePool
- GlobalMaxPool, GlobalAveragePool

### التسوية
- BatchNormalization, InstanceNormalization, LayerNormalization

## أمثلة عملية

### تصنيف صورة

```مرجع
# تحميل النموذج
مصنف = أونكس.حمّل_مسبق("resnet50")

# تحميل صورة وتحويلها
صورة = حمّل_صورة("cat.jpg")
موتر = صورة.إلى_موتر([1, 3, 224, 224])

# التصنيف
نتيجة = مصنف.صنّف(موتر)
اطبع("الفئة:", نتيجة.فئة, "الثقة:", نتيجة.ثقة)
```

### شبكة عصبية بسيطة

```مرجع
# إنشاء شبكة لـ MNIST
شبكة = أونكس.شبكة_جديدة("mnist_classifier")

شبكة.أضف_طبقة("كثيف", 784, 256, تنشيط="ريلو")
شبكة.أضف_طبقة("إسقاط", 0.2)
شبكة.أضف_طبقة("كثيف", 256, 128, تنشيط="ريلو")
شبكة.أضف_طبقة("كثيف", 128, 10, تنشيط="سوفت_ماكس")

# تصدير
شبكة.صدّر("mnist.onnx")
```

```مرجع
# استخدام FP16 للسرعة
إعدادات = أونكس.إعدادات(دقة="عائم_16")
```

## التكامل مع Python

يمكن تصدير النماذج من PyTorch:

```python
# Python
import torch
import torch.onnx

model = MyModel()
dummy_input = torch.randn(1, 3, 224, 224)
torch.onnx.export(model, dummy_input, "model.onnx")
```

ثم استخدامها في المرجع:

```مرجع
# لغة المرجع
نموذج = أونكس.حمّل("model.onnx")
```

## استكشاف الأخطاء

### خطأ: النموذج غير موجود
```مرجع
# التحقق من وجود الملف
إذا ليس ملف.موجود("model.onnx"):
    اطبع("الملف غير موجود!")
```

### خطأ: الشكل غير متوافق
```مرجع
# التحقق من شكل المدخلات
اطبع("المطلوب:", نموذج.معلومات.مداخلات[0].شكل)
اطبع("المقدم:", موتر.شكل)
```
