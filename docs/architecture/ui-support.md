## 📋 نظرة عامة

نظام واجهات المستخدم في لغة المرجع هو إطار عمل متكامل لبناء تطبيقات واجهات المستخدم باستخدام كلمات مفتاحية عربية بسيطة وأنيقة. يوفر النظام مجموعة شاملة من الميزات التي تجعل بناء الواجهات سهلاً وممتعاً.

### الميزات الرئيسية

| الميزة | الوصف | الأولوية |
|--------|-------|----------|
| **نظام التخطيط التلقائي** | صف، عمود، شبكة، مرن | 🔴 حرج |
| **ربط البيانات التلقائي** | تحديث الواجهة تلقائياً | 🔴 حرج |
| **المكونات القابلة لإعادة الاستخدام** | بناء مكونات مخصصة | 🔴 حرج |
| **التصميم المتجاوب** | تكييف حسب حجم الشاشة | 🟡 مهم |
| **نظام الثيمات** | تخصيص الألوان والخطوط | 🟡 مهم |
| **الرسوم البيانية** | رسوم بيانية تفاعلية | 🟡 مهم |
| **النوافذ المنبثقة** | حوارات وتنبيهات | 🟡 مهم |

## 📐 نظام التخطيط

### 1. الصف (Row)

الصف يرتب العناصر أفقياً من اليمين إلى اليسار (RTL) أو العكس.

```almarjaa
صف {
    فجوة: 10،                  // المسافة بين العناصر
    محاذاة: "وسط"،             // محاذاة عمودية
    تبرير: "بين"،              // توزيع أفقي
    حشو: 15،                   // الحشو الداخلي
    
    زر("السابق")،
    نص("الصفحة 1 من 10")،
    زر("التالي")،
}
```

**خصائص الصف:**

| الخاصية | النوع | القيمة الافتراضية | الوصف |
|---------|-------|-------------------|-------|
| `فجوة` | رقم | 0 | المسافة بين العناصر |
| `محاذاة` | نص | "بداية" | المحاذاة العمودية |
| `تبرير` | نص | "بداية" | التوزيع الأفقي |
| `حشو` | رقم | 0 | الحشو الداخلي |
| `هامش` | رقم | 0 | الهامش الخارجي |

### 2. العمود (Column)

العمود يرتب العناصر عمودياً من الأعلى إلى الأسفل.

```almarjaa
عمود {
    فجوة: 15،
    محاذاة: "بداية"،          // محاذاة أفقية
    
    نص("العنوان الرئيسي") {
        حجم: 24،
        وزن_الخط: "غامق"،
    }،
    
    نص("هذا وصف تفصيلي للعنوان الرئيسي...") {
        لون: "#666666"،
    }،
    
    صف {
        زر("إقرأ المزيد")،
        زر("شارك") { نوع: "ثانوي" }،
    }،
}
```

### 3. الشبكة (Grid)

الشبكة توفر تخطيطاً ثنائي الأبعاد مرناً.

```almarjaa
شبكة {
    أعمدة: 3،                  // عدد الأعمدة
    صفوف: "تلقائي"،            // عدد الصفوف
    فجوة_أفقية: 20،
    فجوة_عمودية: 15،
    
    // العناصر
    بطاقة_منتج("منتج 1"، 100)،
    بطاقة_منتج("منتج 2"، 150)،
    بطاقة_منتج("منتج 3"، 200)،
    بطاقة_منتج("منتج 4"، 120)،
    بطاقة_منتج("منتج 5"، 180)،
    بطاقة_منتج("منتج 6"، 90)،
}
```

**تخصيص الشبكة:**

```almarjaa
شبكة {
    مسارات_أعمدة: ["1فر"، "2فر"، "1فر"]،  // أعمدة بنسب متفاوتة
    مسارات_صفوف: ["تلقائي"، "1فر"]،        // صفوف مرنة
    
    بطاقة { عمود: 1، صف: 1، تمديد_عمود: 2 }،
    بطاقة { عمود: 3، صف: 1 }،
    بطاقة { عمود: 1، صف: 2، تمديد_أعمدة: 3 }،
}
```

### 4. الحاوية المرنة (FlexContainer)

```almarjaa
مرن {
    اتجاه: "عمودي"،           // أفقي/عمودي
    التفاف: "التفاف"،          // التفاف العناصر
    تبرير: "موزع"،            // توزيع العناصر
    محاذاة: "وسط"،            // محاذاة العناصر
    
    زر("زر 1")،
    زر("زر 2")،
    زر("زر 3")،
}
```

## 🔗 ربط البيانات

### القيم القابلة للملاحظة (Observable)

```almarjaa
// إنشاء قيمة قابلة للملاحظة
متغير العدد = قابل_للملاحظة(0)؛

// قراءة القيمة
اطبع(العدد.قيمة)؛    // 0

// تعديل القيمة - سيتم إشعار جميع المراقبين
العدد.قيمة = 10؛
```

### أنواع الربط

#### 1. الربط الأحادي (One-Way)

```almarjaa
// البيانات تتدفق من المصدر إلى الواجهة فقط
نص(العدد.قيمة) {
    // النص يتحدث تلقائياً عند تغيير العدد
}
```

#### 2. الربط الثنائي (Two-Way)

```almarjaa
// البيانات تتدفق في الاتجاهين
إدخال {
    ربط: العدد.قيمة،
    // تغيير الإدخال يحدث القيمة
    // وتغيير القيمة يحدث الإدخال
}
```

#### 3. الربط لمرة واحدة (One-Time)

```almarjaa
// القيمة تُقرأ مرة واحدة فقط
نص(العدد.قيمة) {
    ربط: "مرة_واحدة"،
}
```

### القيم المحسوبة (Computed)

```almarjaa
// قيمة تُحسب تلقائياً من قيم أخرى
متغير المجموع = محسوب(() => {
    أرجع العدد.قيمة * 2 + 10؛
})؛

// المجموع يتحدث تلقائياً عند تغيير العدد
نص(المجموع.قيمة)؛
```

### المراقبة (Watcher)

```almarjaa
// مراقبة التغييرات وتنفيذ إجراءات
راقب(العدد، (القيمة_الجديدة، القيمة_القديمة) => {
    اطبع("تغير العدد من " + القيمة_القديمة + " إلى " + القيمة_الجديدة)؛
})؛
```

### قائمة المكونات المتاحة

| المكون | الوصف | الاستخدام |
|--------|-------|-----------|
| `زر` | زر قابل للنقر | إجراءات المستخدم |
| `نص` | عرض النصوص | عناوين، فقرات |
| `إدخال` | حقل إدخال نص | نماذج الإدخال |
| `منطقة_نص` | منطقة نص متعدد الأسطر | تعليقات، رسائل |
| `خانة` | خانة اختيار | خيارات متعددة |
| `راديو` | زر راديو | خيار واحد من عدة |
| `اختيار` | قائمة منسدلة | اختيار من قائمة |
| `منزلق` | شريط تمرير | قيم ضمن نطاق |
| `تقدم` | شريط التقدم | عرض التقدم |
| `مؤقت` | مؤشر تحميل | انتظار |
| `بطاقة` | حاوية بظل | تجميع محتوى |
| `قائمة` | قائمة عناصر | عرض قوائم |
| `جدول` | جدول بيانات | عرض بيانات جدولية |
| `نموذج` | حاوية نموذج | جمع بيانات |
| `تسمية` | تسمية نصية | وصف حقول |
| `صورة` | عرض صورة | محتوى مرئي |
| `أيقونة` | أيقونة | رموز |
| `شارة` | شارة تنبيه | إشعارات |
| `صورة_شخصية` | صورة شخصية | ملفات تعريف |
| `تلميح` | تلميح أدوات | مساعدة |

### إنشاء مكون مخصص

```almarjaa
// تعريف مكون مخصص
مكون بطاقة_منتج {
    // الخصائص المدخلة
    خصائص: {
        الاسم: نص،
        السعر: رقم،
        الصورة: نص،
        مخفض: منطقي = خطأ،
    }،
    
    // الحالة الداخلية
    حالة: {
        محدد: منطقي = خطأ،
    }،
    
    // دورة الحياة
    عند_الإنشاء: () => {
        اطبع("تم إنشاء البطاقة")؛
    }،
    
    // القالب
    بطاقة {
        ظل: "صغير" + (محدد ? " كبير" : "")،
        نقر: () => { محدد = ليس محدد؛ }،
        
        صف {
            صورة(الصورة) {
                عرض: 80،
                ارتفاع: 80،
                حدود_مستديرة: 8،
            }،
            
            عمود {
                فجوة: 5،
                
                نص(الاسم) {
                    وزن_الخط: "غامق"،
                    حجم: 16،
                }،
                
                إذا مخفض {
                    نص(السعر * 0.8 + " ر.س") {
                        لون: "#e74c3c"،
                        وزن_الخط: "غامق"،
                    }،
                    نص(السعر + " ر.س") {
                        لون: "#999"،
                        خط_عبر: صحيح،
                        حجم: 12،
                    }،
                } وإلا {
                    نص(السعر + " ر.س") {
                        لون: "#27ae60"،
                        وزن_الخط: "غامق"،
                    }،
                }،
            }،
        }،
    }،
}

// استخدام المكون
بطاقة_منتج {
    الاسم: "هاتف ذكي"،
    السعر: 2500،
    الصورة: "phone.png"،
    مخفض: صحيح،
}
```

## 📱 التصميم المتجاوب

### نقاط التوقف (Breakpoints)

| النقطة | العرض | الجهاز |
|--------|-------|--------|
| `هاتف` | 0 - 576px | الهواتف |
| `لوحي` | 576px - 992px | الأجهزة اللوحية |
| `حاسوب` | 992px - 1200px | الحواسيب الصغيرة |
| `كبير` | 1200px - 1400px | الحواسيب المتوسطة |
| `ضخم` | 1400px+ | الحواسيب الكبيرة |

### القيم المتجاوبة

```almarjaa
// قيم مختلفة حسب حجم الشاشة
متغير الأعمدة = متجاوب({
    هاتف: 1،      // عمود واحد على الهاتف
    لوحي: 2،      // عمودان على اللوحي
    حاسوب: 4،     // 4 أعمدة على الحاسوب
})؛

شبكة {
    أعمدة: الأعمدة.قيمة،
    // تتكيف تلقائياً
}
```

### إخفاء/إظهار حسب الجهاز

```almarjaa
صف {
    // يظهر على الهواتف فقط
    قائمة_الجوال { 
        إظهار_على: "هاتف"،
    }،
    
    // يظهر على الحاسوب فقط
    قائمة_سطح_المكتب { 
        إظهار_على: ["حاسوب"، "كبير"، "ضخم"]،
    }،
    
    // يُخفى على الهواتف
    الشريط_الجانبي {
        إخفاء_على: "هاتف"،
    }،
}
```

### استعلامات الوسائط

```almarjaa

    أزرار: [
        { نص: "إلغاء"، نوع: "ثانوي"، قيمة: "إلغاء" }،
        { نص: "حذف"، نوع: "خطر"، قيمة: "حذف" }،
    ]،
}؛

إذا النتيجة == "حذف" {
    احذف_العنصر()؛
}
```

### الحركة الأساسية

```almarjaa
حرك(العنصر) {
```

| `سهولة_خروج` | بطيء في النهاية |
| `سهولة_دخول_خروج` | بطيء في البداية والنهاية |
| `ارتداد` | تأثير الارتداد |

## 📖 أمثلة شاملة

### مثال: تطبيق قائمة مهام

```almarjaa
// تطبيق قائمة مهام بسيط

// الحالة
متغير المهام = قابل_للملاحظة([])؛
متغير النص_الجديد = قابل_للملاحظة("")؛

// إضافة مهمة
دالة أضف_مهمة() {
    إذا النص_الجديد.قيمة != "" {
        المهام.قيمة.أضف({
            نص: النص_الجديد.قيمة،
            مكتمل: خطأ،
        })؛
        النص_الجديد.قيمة = ""؛
    }
}

// تبديل حالة المهمة
دالة تبديل(الفهرس) {
    المهام.قيمة[الفهرس].مكتمل = ليس المهام.قيمة[الفهرس].مكتمل؛
}

// الواجهة
عمود {
    فجوة: 20،
    حشو: 20،
    
    // العنوان
    نص("قائمة المهام") {
        حجم: 24،
        وزن_الخط: "غامق"،
    }،
    
    // إدخال مهمة جديدة
    صف {
        فجوة: 10،
        
        إدخال {
            ربط: النص_الجديد.قيمة،
            placeholder: "أضف مهمة جديدة..."،
            عند_الإرسال: أضف_مهمة،
            تمدد: صحيح،
        }،
        
        زر("إضافة") {
            نقر: أضف_مهمة،
            لون_الخلفية: الثيم.الأساسي،
        }،
    }،
    
    // قائمة المهام
    قائمة {
        بيانات: المهام.قيمة،
        
        عنصر: (مهمة، فهرس) => {
            صف {
                فجوة: 10،
                
                خانة {
                محدد: مهمة.مكتمل،
                    تغيير: () => تبديل(فهرس)،
                }،
                
                نص(مهمة.نص) {
                    خط_عبر: مهمة.مكتمل،
                    لون: مهمة.مكتمل ? "#999" : "#333"،
                }،
            }،
        }،
    }،
}
```
