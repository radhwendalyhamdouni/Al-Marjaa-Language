# نظام الحزم الكامل - Package Manager

**الإصدار:** 3.3.0  
**آخر تحديث:** 2026

---

## نظرة عامة

نظام حزم متكامل ومتقدم للغة المرجع يدعم:

- **Registry مركزي**: مستودع رسمي للحزم
- **نشر الحزم**: أمر `انشر` لنشر الحزم
- **حل التبعيات**: حل ذكي للتبعيات
- **Lock file**: قفل الإصدارات
- **التوزيع الثنائي**: تحميل ملفات ثنائية جاهزة
- **توقيع الحزم**: أمان وتحقق من المصدر
- **Workspace**: دعم المشاريع المتعددة

## 1. الأوامر الأساسية

### 1.1 تهيئة مشروع جديد

```almarjaa
# إنشاء مشروع جديد
مرجع أنشئ مشروع اسم:"حاسبة"

# النتيجة:
# ├── مشروع.toml
# ├── مصدر/
# │   └── رئيسي.mrj
# └── حزم/
```

### 1.2 إضافة وإزالة الحزم

```almarjaa
# إضافة حزمة
مرجع أضف "json" إصدار:"1.2.0"
مرجع أضف "http"

# إزالة حزمة
مرجع أزل "json"

# تثبيت كل الحزم
مرجع ثبّت
```

### 1.3 البحث والمعلومات

```almarjaa
# البحث عن حزمة
مرجع ابحث "web"

# معلومات حزمة
مرجع معلومات "json"

# قائمة الحزم المثبتة
مرجع قائمة
```

### 1.4 التحديث والتنظيف

```almarjaa
# تحديث الحزم
مرجع حدّث
مرجع حدّث ["json", "http"]

# التحقق من التحديثات
مرجع تحقق

# تنظيف الحزم غير المستخدمة
مرجع نظّف
```

### 1.5 النشر

```almarjaa
# نشر حزمة
مرجع انشر

# نشر مع توقيع
مرجع انشر موقّع
```

```toml

[تبعيات]

اختبارات = "2.0.0"

كامل = ["json", "http", "websocket"]
```

## 4. نظام Workspace

### 4.1 هيكل المشروع

```
مشروع_كبير/
├── مساحة.toml
├── حزم/
│   ├── نواة/
│   │   └── مشروع.toml
│   ├── واجهة/
│   │   └── مشروع.toml
│   └── أدوات/
│       └── مشروع.toml
└── تطبيق/
    └── مشروع.toml
```

### 4.2 ملف التكوين

```toml
# مساحة.toml
[مساحة]
اسم = "مشروع_كبير"

[أعضاء]
حزم = ["نواة", "واجهة", "أدوات"]
تطبيق = ["تطبيق"]

[تبعيات.مشتركة]
json = "1.2.0"

[تكوين]
رفع_التبعيات = صحيح
بناء_متوازي = صحيح
```

### 4.3 أوامر Workspace

```almarjaa
# بناء كل الحزم
مرجع مساحة ابنِ

# اختبار كل الحزم
مرجع مساحة اختبر

# نشر كل الحزم
مرجع مساحة انشر
```

## 5. التوزيع الثنائي

### 5.1 تحميل الملفات الثنائية

```almarjaa
# تحميل أحدث إصدار
مرجع حمّل
```

### 5.2 المنصات المدعومة

| النظام | المعمارية | الملف |
|--------|----------|-------|
| Linux | x86_64 | almarjaa-linux-x86_64 |
| Linux | aarch64 | almarjaa-linux-aarch64 |
| macOS | x86_64 | almarjaa-macos-x86_64 |
| macOS | aarch64 | almarjaa-macos-aarch64 |
| Windows | x86_64 | almarjaa-windows-x86_64.exe |

### 5.3 التحقق من التحديثات

```almarjaa
مرجع تحقق_تحديث

```

## 6. توقيع الحزم

### 6.1 إنشاء مفتاح

```almarjaa
# إنشاء مفتاح توقيع جديد
مفتاح = أنشئ_مفتاح("أحمد محمد", "ahmed@example.com")

# عرض معرف المفتاح
اطبع(mفتاح.معرف)  # abc123def456
```

### 6.2 توقيع حزمة

```almarjaa
# توقيع حزمة
توقيع = وقّع_حزمة("حاسبة", "1.0.0", المفتاح, المحتوى)

# التحقق من التوقيع
إذا تحقق_من_التوقيع(التوقيع, المفتاح, المحتوى) {
    اطبع("التوقيع صالح")
}
```

### 6.3 إدارة المفاتيح الموثوقة

```almarjaa
# إضافة مفتاح موثوق
سجل.أضف_مفتاح(المفتاح)
سجل.وثّق_مفتاح(المفتاح.معرف)

# التحقق من الثقة
إذا سجل.موثوق(المفتاح.معرف) {
    اطبع("المفتاح موثوق")
}
```

## 7. الأمان

### 7.1 التحقق الأمني

```almarjaa
# تقرير الأمان
مرجع أمان

# النتيجة:
# ✅ json v1.2.0 - آمن
# ⚠️ http v0.5.0 - ثغرة متوسطة
# ❌ old v0.1.0 - ثغرة حرجة
```

### 7.2 مستويات الخطورة

| المستوى | الوصف | الإجراء |
|---------|-------|---------|
| منخفض | لا يشكل خطراً | تحذير فقط |
| متوسط | ثغرة قابلة للاستغلال | تحديث موصى به |
| عالي | ثغرة حرجة | تحديث إجباري |
| حرج | خطر فوري | منع التثبيت |

## 8. الإصدارات

### 8.1 الإصدار الدلالي (SemVer)

```
MAJOR.MINOR.PATCH

MAJOR: تغييرات غير متوافقة
MINOR: ميزات جديدة متوافقة
PATCH: إصلاحات
```

### 8.2 تحديد الإصدار

```toml
# إصدار محدد
json = "1.2.0"

# نطاق إصدار
json = "^1.2.0"   # 1.x.x مع x >= 2
json = "~1.2.0"   # 1.2.x
json = ">=1.0.0"  # أي إصدار >= 1.0.0

# أحدث إصدار
json = "*"
json = "latest"
```

## 11. التكامل المستمر (CI)

### 11.1 GitHub Actions

```yaml
name: نشر الحزمة

on:
  push:
    tags:
      - 'v*'

jobs:
  publish:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: تثبيت المرجع
        run: curl -fsSL https://almarjaa.io/install.sh | sh
      
      - name: نشر الحزمة
        run: مرجع انشر
        env:
          ALMARJAA_TOKEN: ${{ secrets.ALMARJAA_TOKEN }}
```

## 12. أمثلة متقدمة

### 12.1 مشروع كامل

```toml
# مشروع.toml
[مشروع]
اسم = "خادم_ويب"
إصدار = "2.0.0"
وصف = "خادم ويب عربي متقدم"
مؤلف = "فريق المرجع"
رخصة = "MIT"
كلمات_مفتاحية = ["web", "server", "arabic"]

[تبعيات]
http = "^1.0.0"
json = "^2.0.0"
crypto = "^1.5.0"

[تبعيات.تطوير]
اختبارات = "^1.0.0"
توثيق = "^0.5.0"

[نقاط_الدخول]
رئيسي = "مصدر/رئيسي.mrj"
cli = "مصدر/cli.mrj"

[ميزات]
افتراضي = ["http", "json"]
tls = ["crypto", "tls_extra"]
websocket = ["websocket_core"]
```

```toml

[مساحة]

[أعضاء]

    "حزم/نواة",
    "حزم/واجهة", 
    "حزم/أدوات",
    "حزم/اختبارات"
]

[تبعيات.مشتركة]
json = "2.0.0"
logging = "1.0.0"

[تكوين]
رفع_التبعيات = صحيح
فرض_الإصدارات = صحيح
بناء_متوازي = صحيح
```
