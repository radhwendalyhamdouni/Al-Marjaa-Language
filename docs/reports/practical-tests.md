## 🤖 Vibe Coding

### ما هو Vibe Coding؟
Vibe Coding هو أسلوب برمجة يسمح لك بكتابة النص العربي الطبيعي وتحويله تلقائياً إلى كود تنفيذي.

### كيفية الاستخدام:

```bash
# الطريقة 1: من REPL
./target/release/almarjaa --repl
# ثم اكتب: ذكاء

# الطريقة 2: باستخدام API
# (يتطلب تحميل نموذج AI)
```

### تحميل نموذج AI:

```bash
mkdir -p models
wget -O models/qwen2.5-0.5b-instruct-q4_k_m.gguf \
  'https://huggingface.co/Qwen/Qwen2.5-0.5B-Instruct-GGUF/resolve/main/qwen2.5-0.5b-instruct-q4_k_m.gguf'
```

## 🐛 المشاكل المكتشفة

### 1. العودية العميقة
- **المشكلة**: stack overflow مع العودية العميقة
- **الحل**: زيادة حجم المكدس أو استخدام التكرار

### 2. بعض الأمثلة القديمة
- **المشكلة**: بعض الأمثلة تستخدم صيغة قديمة
- **الحل**: تحديث الأمثلة للصيغة الجديدة

### 3. أمثلة Vibe Coding
- **المشكلة**: ملفات vibe_*.mrj هي أكواد Rust
- **الحل**: إنشاء أمثلة .mrj صحيحة
