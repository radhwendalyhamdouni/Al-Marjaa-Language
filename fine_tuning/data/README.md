# 📊 بيانات التدريب

## 📥 تحميل البيانات

يمكنك إنشاء بيانات التدريب باستخدام:

```bash
python3 ../prepare_training_data.py
```

## 📋 الملفات المتوقعة

- `train_chat.jsonl` - بيانات التدريب (صيغة Chat)
- `train_alpaca.jsonl` - بيانات التدريب (صيغة Alpaca)
- `eval_chat.jsonl` - بيانات التقييم (صيغة Chat)
- `eval_alpaca.jsonl` - بيانات التقييم (صيغة Alpaca)
