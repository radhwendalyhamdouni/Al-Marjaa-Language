# 🏋️ نظام التدريب - Fine-Tuning System

هذا المجلد يحتوي على أدوات تدريب نموذج الذكاء الاصطناعي.

---

## 📁 البنية

```
fine_tuning/
├── data/              # بيانات التدريب
│   ├── train_*.jsonl  # ملفات التدريب
│   └── eval_*.jsonl   # ملفات التقييم
└── scripts/           # سكربتات التدريب
```

---

## 🚀 الاستخدام

```bash
# 1. تجهيز بيانات التدريب
python3 fine_tuning/prepare_training_data.py

# 2. بدء التدريب
python3 fine_tuning/train_lora.py

# 3. اختبار النموذج المُدرّب
python3 fine_tuning/test_finetuned.py
```
