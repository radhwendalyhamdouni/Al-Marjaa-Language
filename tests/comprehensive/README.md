## 🚀 تشغيل الاختبارات

### جميع الاختبارات
```bash
cargo test --all-features
```

### اختبارات محددة
```bash
# اختبارات المحلل المعجمي
cargo test lexer_comprehensive

# اختبارات المفسر
cargo test interpreter_comprehensive

# اختبارات AI
cargo test ai_integration

# اختبارات LSP
cargo test lsp_server
```

### مع التغطية
```bash
cargo tarpaulin --out Html --output-dir coverage/
```

## 📊 فئات الاختبارات

### 1. اختبارات المحلل المعجمي (Lexer)

| الفئة | الوصف |
|-------|-------|
| الأرقام | أرقام عربية، إنجليزية، ست عشرية، ثنائية |
| النصوص | نصوص عربية، تسلسلات الهروب، نصوص متعددة الأسطر |
| الكلمات المفتاحية | جميع الكلمات المحجوزة العربية والإنجليزية |
| المعاملات | حسابية، مقارنة، إسناد، منطقية |
| الفواصل | عربية وإنجليزية |
| التعليقات | سطر واحد ومتعددة الأسطر |
| الأداء | سرعة التحليل المعجمي |

### 2. اختبارات المفسر (Interpreter)

| الفئة | الوصف |
|-------|-------|
| المتغيرات | تعريف، إسناد، متعددة |
| العمليات الحسابية | جمع، طرح، ضرب، قسمة، أس، باقي |
| المقارنات | مساوٍ، غير مساوٍ، أكبر، أصغر |
| المنطق | و، أو، ليس |
| الجمل الشرطية | إذا، وإلا، وإذا |
| الحلقات | طالما، لكل، كرر |
| الدوال | تعريف، استدعاء، عودية، لامدا |
| القوائم | إنشاء، وصول، قوائم استيعابية |
| القواميس | إنشاء، وصول |
| الفئات | تعريف، وراثة، طرق |
| الأخطاء | حاول، امسك، ألقِ |
| التفكيك | قوائم وكائنات |
| الأداء | سرعة التنفيذ |

### 3. اختبارات تكامل AI

| الفئة | الوصف |
|-------|-------|
| Tensor | إنشاء، عمليات، تحويل |
| NeuralNetwork | إنشاء، طبقات، تدريب |
| التدريب | محسنات، خسارة، epochs |
| Autograd | حساب التدرجات |
| DataLoader | تحميل دفعات البيانات |
| GPU | عمليات على GPU |
| Regularization | Dropout, BatchNorm, L2 |
| الأداء | سرعة العمليات |

### 4. اختبارات LSP Server

| الفئة | الوصف |
|-------|-------|
| التحليل المعجمي | tokens، أخطاء |
| التحليل النحوي | AST، استرداد الأخطاء |
| الـ Linter | تحذيرات، أخطاء |
| التكامل | خط المعالجة الكامل |
| الموقع | تتبع الأسطر والأعمدة |
| الإكمال | الكلمات المفتاحية |
| الأداء | سرعة التحليل |
| العربية | أرقام، فواصل، معاملات |
