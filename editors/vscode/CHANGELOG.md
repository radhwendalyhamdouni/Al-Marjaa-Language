# Changelog

All notable changes to the **Al-Marjaa Language** extension will be documented in this file.

## [2.0.0] - 2025-03-07

### ✨ Added - إضافات جديدة

#### Vibe Coding Integration
- 🤖 **Vibe Coding**: تكامل AI ذكي لشرح وتحسين الكود
- 💡 **Inline Completion**: اقتراحات تلقائية أثناء الكتابة
- ✨ **Code Actions**: إجراءات سريعة (شرح، تحسين، توليد اختبارات)

#### Code Lens
- ▶️ زر تشغيل لكل دالة
- 🧪 زر اختبار لكل دالة
- ℹ️ زر شرح لكل دالة

#### New Providers
- 📍 **Definition Provider**: الانتقال للتعريف (F12)
- 🔗 **Reference Provider**: إيجاد المراجع (Shift+F12)
- ✏️ **Rename Provider**: إعادة تسمية ذكية (F2)
- 🎨 **Document Formatting**: تنسيق تلقائي
- 🎨 **Color Provider**: عرض الألوان
- 📁 **Folding Provider**: طي الأكواد
- 🔍 **Document Symbol Provider**: قائمة الرموز

#### Themes
- 🌙 **Al-Marjaa Dark**: ثيم داكن احترافي
- 🌃 **Al-Marjaa Arabic Night**: ثيم ليلي عربي

#### Commands
- ✨ `almarjaa.explainCode` - شرح الكود
- ⚡ `almarjaa.optimizeCode` - تحسين الكود
- 🧪 `almarjaa.generateTests` - توليد اختبارات
- 🔄 `almarjaa.convertToTypeScript` - تحويل إلى TypeScript
- 🐍 `almarjaa.convertToPython` - تحويل إلى Python
- 📊 `almarjaa.showStats` - إحصائيات الكود

#### Snippets
- 35+ snippet جديدة
- دعم الدوال غير المتزامنة
- دعم الـ Decorators
- دعم الـ Pattern Matching

### 🔧 Changed - تغييرات
- تحسين الأداء العام
- تحسين تلوين الصيغة
- تحسين الإكمال التلقائي

### 🐛 Fixed - إصلاحات
- إصلاح مشاكل الترميز العربي
- إصلاح تعطل الخادم في بعض الحالات
